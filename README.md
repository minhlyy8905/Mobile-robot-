1. Autonomous Mobile Robot using ROS 2 and Jetson Nano

An autonomous mobile robot developed on ROS 2 and NVIDIA Jetson Nano platform. The system integrates computer vision, LiDAR sensing, and multiple control methods for intelligent navigation and obstacle avoidance.

2, Features
Autonomous navigation using ROS 2.
Traffic sign detection and recognition using AI-based vision models.
Real-time obstacle detection with LiDAR.
Camera-based environmental perception.
Manual control via keyboard, mouse, and voice commands.
Autonomous movement based on detected traffic signs.
Real-time sensor data processing and visualization.
